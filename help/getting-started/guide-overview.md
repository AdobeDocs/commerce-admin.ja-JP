# Adobe Commerce入門ガイド

このガイドは、Adobe Commerce管理者が初めて利用するマーチャントおよびシステム管理者を対象としています。 それらの視点からプラットフォームの概要を説明し、機能的なストアを実現する基本的な機能に関する詳細情報を提供します。

主な内容：

| 件名 | 説明 |
| ------- | ----------- |
| [&#x200B; ログイン資格情報とURL](login-urls.md) | 管理者およびストアフロントへのアクセス権を取得するために使用される[!DNL Commerce] URLとアカウント資格情報について説明します。 |
| [[!DNL Commerce]  アカウント &#x200B;](commerce-account-create.md) | 購入した製品とサービスを管理する[!DNL Commerce] アカウントについて説明します。 |
| [管理者](admin.md) | 販売者が製品とプロモーションを設定し、注文を管理し、その他の管理タスクを実行する[!DNL Commerce]管理者について説明します。 |
| [&#x200B; ストアフロント &#x200B;](storefront.md) | ストアが顧客のショッピング体験をサポートするために提供できるページと機能要素について説明します。 |
| [基本的な設定と設定](websites-stores-views.md) | ストア階層と設定スコープの概念について説明し、基本的な設定設定について説明します。 |
| [&#x200B; サイトのコンプライアンス &#x200B;](privacy-policy.md) | 安全な環境を維持し、法務の要件とベストプラクティスを満たすための業界ガイドラインを、加盟店がどのように満たしているかを学びましょう。 |
| [標準レポート &#x200B;](reports-menu.md) | データをフィルタリングしてオンラインレポートを生成する方法について説明します。 レポートデータは、スプレッドシートで開くことも、他のアプリケーションに読み込むこともできます。 |
| [&#x200B; リソース &#x200B;](resources.md) | ビジネスや技術リソース、セルフヘルプツール、成功に役立つサービスなど、豊富なリソースについて学びましょう。 |

{style="table-layout:auto"}

Adobe Commerceのより詳細な機能については説明しません。
