# Adobe Commerce エコシステム

Adobeには、ビジネスおよび技術に関する豊富なリソース、セルフヘルプツール、サービスが用意されており、お客様の成功を支援します。

| エコシステム資源 | 説明 |
| ------------------ | ----------- |
| [&#x200B; 製品ドキュメント &#x200B;](https://experienceleague.adobe.com/docs/commerce.html?lang=ja) | マーチャント、デザイナー、デベロッパーは、このドキュメントを使用して、[!DNL Commerce] の機能をビジネスの成長にどのように活用できるかを深くinsightできます。 |
| [Commerce リソースライブラリ &#x200B;](https://business.adobe.com/jp/resources/main.html?Products+%26+Services=Commerce%252CCommerce%2520Cloud) | 電子書籍、Commerceのホワイトペーパー、ウェビナーにアクセスして、insightおよびオンラインビジネスのエキスパート資料を入手し、店舗の開発と改善に役立ててください。 |
| [Adobe製品セキュリティインシデント対応 &#x200B;](https://helpx.adobe.com/jp/security.html) | セキュリティアラートレジストリに参加して、潜在的な脆弱性と [&#x200B; セキュリティのベストプラクティス &#x200B;](https://www.adobe.com/content/dam/cc/en/security/pdfs/Adobe-Magento-Commerce-Best-Practices-Guide.pdf) に関する最新情報を取得してください。 |
| [&#x200B; コミュニティフォーラム &#x200B;](https://community.magento.com/) | このフォーラムでは、Adobe CommerceとMagento Open Sourceに熱心に取り組むユーザーが集まり、ヒント、テクニック、サポートを提供しています。 |
| [&#x200B; ブログ &#x200B;](https://business.adobe.com/jp/blog/) | 新機能、トレンド、ベストプラクティス、今後のイベントなどに関する最新情報については、ブログをご覧ください。 |
| [&#x200B; イベントとウェビナー &#x200B;](https://www.adobe.com/events.html) | 今後のイベントやウェビナーのカレンダーを常に把握します。 [!DNL Commerce] のエコシステムでは常に何かが起きています。 |
| [[!DNL Commerce Marketplace]](https://marketplace.magento.com/) | デジタルコマースソリューションの [!DNL Commerce] 次世代市場をご覧ください。 [!DNL Commerce Marketplace] は、厳選されたソリューションをマーチャントに提供すると同時に、有能な開発者に豊富なツール、プラットフォーム、そしてビジネスを盛り上げる上で最適な場所を提供しています。 |
| [&#x200B; 講習、認定等 &#x200B;](https://learning.adobe.com/catalog.html?solution=Adobe%20Commerce) | Adobe CommerceとMagento Open Sourceの機能と柔軟性を最大限に活用する方法を説明します。 トレーニングは、マーケター、デザイナー、開発者など、ビジネスのあらゆる役割に対して提供されます。 経験豊富な [!DNL Commerce] の専門家は、[!DNL Commerce] の認定を取得することで、実際のスキルを検証できます。 |
| [&#x200B; 組合員 &#x200B;](https://business.adobe.com/jp/products/magento/partners.html) | Adobe パートナーは、お客様の成功に全力で取り組み、オンプレミスおよびクラウドベースのソリューションに対して、カスタム統合、クラス最高のカスタマーエクスペリエンス、戦略的なマーケティングの取り組み、エキスパートによるパフォーマンスとスケーラビリティの最適化を提供します。 |
| [Adobe CommerceManaged Services](https://business.adobe.com/jp/products/magento/fully-managed-service.html) | Adobe Commerce Managed Services チームは、マーチャントおよびソリューションパートナーが成功を最大化するのに役立ちます。 これらのエキスパートは、アーキテクチャの計画からデプロイ後に至るまで、包括的な分析とベストプラクティスの推奨事項を提供します。 |
