# Adobe Commerce マーチャンダイジングとプロモーション ガイド

このガイドは、Adobe CommerceとMagento Open Sourceの管理画面で作業するマーチャンダイザーとマーケティングスタッフを対象としています。 ターゲットを絞った割引やイベントなど、マーチャンダイジングやプロモーション機能に関する詳細な情報を提供します。 コア [!DNL Commerce]の設定と機能に関する基本的な理解を前提としています。

主な内容：

| 件名 | 説明 |
| ------- | ----------- |
| [はじめに](introduction.md) | Commerce インスタンスでのマーチャンダイジング機能とプロモーション機能の概要。 |
| [ マーケティングメニュー](marketing-menu.md) | [!UICONTROL Marketing] メニューを使用して、プロモーション、コミュニケーション、SEO、ユーザー生成コンテンツを管理するための複数の機能にアクセスします。 |
| [ ビジュアルマーチャンダイザー](visual-merchandiser.md) | [!BADGE PaaSのみ]{type=Informative url="https://experienceleague.adobe.com/en/docs/commerce/user-guides/product-solutions" tooltip="Adobe Commerce on Cloud プロジェクト（Adobeで管理されるPaaS インフラストラクチャ）とオンプレミス プロジェクトにのみ適用されます。"}製品を配置し、カテゴリ リストに表示される製品を決定する条件を適用できる高度なツールのセットを探索します。 |
| [ ギフトレジストリ ](gift-registries.md) | お客様に、特別な機会のギフト登録情報を作成したり、友人や家族を招待してギフト登録情報からギフトを購入したりする機能を提供します。 |
| [特典およびロイヤルティプログラム ](rewards-loyalty.md) | 顧客エンゲージメントを促進し、顧客ロイヤルティを向上するプログラムを実施できます。 |
| [個人向け販売とイベント ](events-private-sales.md) | 期間限定セールスの作成、特定のメンバーへのセールスの制限、スタンドアロンのプライベートセールページの作成。 |
| [ カタログ価格ルール ](price-rules-catalog.md) | 定義された条件にもとづいて、バイヤーに商品を割引価格で提供します。 |
| [ カート価格ルール ](price-rules-cart.md) | 一連の条件に基づいて、または顧客が有効なクーポンコードを入力したときに、ショッピングカート内の商品に割引を適用します。 |
| [製品の関係](product-relationships.md) | 製品間の関係の性質を定義することで、他のページでカタログ内の製品を宣伝します。 |
| [ ニュースレターと購読](newsletters.md) | 購読者の顧客にニュースレターを公開および配信して、ストアへのトラフィックを増やします。 |
| [電子メールリマインダー](email-reminder-rules.md) | プロモーションを利用して購入するように、顧客にリマインダーを自動的に送信します。 |
| [検索エンジン最適化](seo-overview.md) | Commerceの機能を実装して、web サイトのコンテンツとプレゼンテーションを調整し、検索エンジンによるインデックスの作成方法を改善します。 |
| [Google サイトツール ](google-tools.md) | Google製品との統合を設定することで、コンテンツの最適化、トラフィックの分析、カタログのショッピングアグリゲーターやマーケットプレイスへの接続が可能になります。 |
