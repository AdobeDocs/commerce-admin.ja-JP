# [!DNL Commerce]のコンテンツとデザインの概要

_コンテンツマーケティング_&#x200B;とは、顧客に有益な情報を無料で提供することで、商品やサービスを宣伝する手法を指します。 コンテンツの質は、ストアを他のストアから区別し、検索エンジンに対する認知度を高め、顧客にサポートを提供するのに役立ちます。 このソフトセルのアプローチは、多くの場合、広告よりも効果的で、信頼性と信頼を築き、ストアを目的地に変えることができます。 コンテンツは引き続き重要。

コンテンツは、ストアのブランディングを反映し、メッセージを伝えるために、独自のビジュアルプレゼンテーションとボイスで配信する必要があります。 ストーリーを伝えるために写真を使う： 有益なインサイトを提供。 質の高いコンテンツを利用すれば、より少ないリソースで済む場合もあることに留意しましょう。

## デザイン

Adobe CommerceとMagento Open Sourceには、ストアを簡単に変更できる、使いやすいデザインオプションが多数用意されています。 [Commerce Marketplace](../getting-started/commerce-marketplace.md)には、プロがデザインしたテーマも多数あります。 ストアのウィンドウドレッシングと同様に、季節やプロモーションごとにテーマを変更できます。

[!BADGE PaaSのみ]{type=Informative url="https://experienceleague.adobe.com/ja/docs/commerce/user-guides/product-solutions" tooltip="Adobe Commerce on Cloud プロジェクト（Adobeで管理されるPaaS インフラストラクチャ）とオンプレミス プロジェクトにのみ適用されます。"}より高度なストア デザイナーは、個別のコンポーネントからページを組み立てるオブジェクト指向の環境で作業する柔軟性を高く評価しています。 基本を理解した後は、そのような柔軟で流動的な環境で作業することに感謝します。 詳しくは、[&#x200B; フロントエンド開発者ガイド &#x200B;](https://developer.adobe.com/commerce/frontend-core/guide/)を参照してください。

### 設定

[&#x200B; デザイン設定](configuration.md)を使用すると、[!DNL Commerce] インストールでweb サイト、ストア、ビューごとに異なる設定を簡単に管理できます。

### テーマ

[&#x200B; テーマ &#x200B;](themes.md)は、ストアの視覚的なプレゼンテーションを決定し、レイアウトファイル、テンプレートファイル、翻訳ファイル、スキンのコレクションで構成されます。

### スケジュール

[!BADGE PaaSのみ]{type=Informative url="https://experienceleague.adobe.com/ja/docs/commerce/user-guides/product-solutions" tooltip="Adobe Commerce on Cloud プロジェクト（Adobeで管理されるPaaS インフラストラクチャ）とオンプレミス プロジェクトにのみ適用されます。"}

テーマは、[&#x200B; スケジュール &#x200B;](schedule.md)に従って、特定の期間にアクティベートできます。 スケジュールを利用して、シーズンやプロモーションのテーマを事前に変更できます。

## コンテンツ

コンテンツは、顧客がストアにアクセスしたときに表示される[&#x200B; ページ &#x200B;](pages.md)と要素を定義します。 テキストや画像などのページの基本的な要素だけでなく、インタラクティブで動的なコンテンツを提供する、より高度な要素を定義して、ショッピング体験を向上させることができます。
