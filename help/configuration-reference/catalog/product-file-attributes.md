## [!UICONTROL Configure Allowed File Types and Size]

![製品ファイル属性](./assets/file-validation.png)

| フィールド | [範囲](../../getting-started/websites-stores-views.md#scope-settings) | 説明 |
|--- |--- |--- |
| [!UICONTROL Allowed File Extensions] | グローバル | `pdf,doc,docx,txt`など、許可されているファイル形式のコンマ区切りリスト。 デフォルトで許可されているファイルタイプ（pdf、doc、docx、xls、xlsx、ppt、pptx、txt、csv、zip）を使用するには、空白のままにします。 |
| [!UICONTROL Maximum File Size] | グローバル | ファイルの最大許容サイズ （MB） （例：`20.0`）。 デフォルトの制限は16 MBです。 許可される最大ファイルサイズは31 MBです。 |
