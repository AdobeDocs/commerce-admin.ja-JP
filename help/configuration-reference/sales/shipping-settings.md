![&#x200B; 接触チャネル &#x200B;](./assets/shipping-settings-origin.png)<!-- zoom -->

|--- |--- |--- |

| [!UICONTROL Region/State] | Web サイト | 原点の領域または状態。 |

| [!UICONTROL City] | Web サイト | 原点都市。 |
| [!UICONTROL Street Address] | Web サイト | POI の住所。 |
