# 設定リファレンスガイド

このガイドは、Adobe CommerceまたはMagento Open Source Admin で作業するマーチャント管理者およびシステム管理者を対象としています。 **[!UICONTROL Stores]**/_[!UICONTROL Settings]_/**[!UICONTROL Configuration]**の_ 管理 _サイドバーからアクセスできるすべてのストア設定の参照情報を提供します。

Adobe CommerceおよびMagento Open Sourceの機能やストア設定の手順の詳細については説明していません。

このガイドは、設定左側のナビゲーションに従って整理されています。
