# 製品設定 – [!UICONTROL Product Reviews]

_[!UICONTROL Product Reviews]_セクションには、顧客が製品について送信したすべてのレビューが一覧表示されます。 このセクションは、新しい製品が初めて保存された後にのみ、他の製品情報と共に表示されます。 詳しくは、[製品レビュー](../merchandising-promotions/product-reviews.md)を参照してください。

![製品レビュー](./assets/product-review.png){width="600" zoomable="yes"}

## フィールドリファレンス

| フィールド | 説明 |
|--- |--- |
| [!UICONTROL ID] | 製品レビューエントリ用に生成された一意の数値ID |
| [!UICONTROL Created] | レビューの公開日 |
| [!UICONTROL Status] | 審査状況（`Pending`、`Approved`または`Not Approved`） |
| [!UICONTROL Title] | タイトルを見る |
| [!UICONTROL Nickname] | レビューを残したユーザーのニックネーム |
| [!UICONTROL Review] | 現在使用している製品のカスタマーレビュー |
| [!UICONTROL Visibility] | ストアレビューでの可視性 |
| [!UICONTROL Type] | レビューを残したユーザーのタイプ （`Guest`または`Customer`） |
| [!UICONTROL Product] | レビュー済みの製品名 |
| [!UICONTROL SKU] | 製品に割り当てられている一意の在庫保管単位 |
| [!UICONTROL Action] | 製品を編集モードで開きます |

{style="table-layout:auto"}

## 特定の製品に対する中程度のレビュー

1. _管理者_ サイドバーで、**[!UICONTROL Catalog]** > **[!UICONTROL Products]**&#x200B;に移動します。

1. 製品を見つけて、編集モードで開きます。

1. _[!UICONTROL Product Reviews]_セクションまでスクロールします。

1. **[!UICONTROL Edit]**&#x200B;をクリックすると、ステータスが`Pending`の製品レビューが表示され、詳細を表示および編集できます。

1. レビュー用にステータスを設定：

   - 保留中のレビューを承認するには、`Approved`を選択します。
   - レビューを却下するには、`Not Approved`を選択します。
   - レビューステータスはいつでも`Pending`に戻すことができます。

1. 完了したら、**[!UICONTROL Save Review]**&#x200B;をクリックします。

`Pending`と`Not Approved`のステータスを持つレビューは、ストアフロントに表示されません。
