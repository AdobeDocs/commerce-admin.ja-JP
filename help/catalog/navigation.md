# カタログナビゲーション

_ナビゲーション_&#x200B;という用語は、買い物客がストア全体を通じてページ間を移動するために使用する方法を指します。 ストアのメインメニュー、つまりトップナビゲーションは、実際にはカテゴリーリンクのリストであり、カタログ内の商品に簡単にアクセスできます。 パンくずリストには、ほとんどのページの上部に表示されるカテゴリと、一部の2列または3列ページの左側に表示される階層化されたナビゲーションにもカテゴリがあります。 カテゴリの表示オプションについて詳しくは、[表示設定](categories-display-settings.md)を参照してください。

ストアで商品を表示するには、少なくとも1つのカテゴリに割り当てる必要があります（[上位ナビゲーションの設定](navigation-top.md)を参照）。 各カテゴリーには、画像、静的ブロック、説明、カテゴリー内の商品リストを含む専用のランディングページを用意することができます。 また、休日やプロモーションなど、特定の期間のみアクティブなカテゴリーページ用の特別なデザインを作成することもできます。
