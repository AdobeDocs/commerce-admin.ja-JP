# Adobe Commerce カタログ管理ガイド

このガイドは、Adobe CommerceとMagento Open Sourceの管理画面で作業する商品マーチャンダイザーと在庫管理者向けに作成されています。 カタログのナビゲーション構造の構築など、製品カタログの機能に関する詳細な情報を提供します。 コア [!DNL Commerce]の設定と機能に関する基本的な理解を前提としています。

主な内容：

| 件名 | 説明 |
| ------- | ----------- |
| [はじめに](introduction.md) | カタログ管理機能を使用して、顧客がストア内の商品をどのように見つけるかを反映する概要。 |
| [ カタログ メニュー](catalog-menu.md) | [!UICONTROL Catalog] メニューを使用して、ストアのカタログ、カテゴリ、製品を管理するための複数の機能にアクセスします。 |
| [ カタログと製品のURL](catalog-urls.md) | 基本的なURL タイプと、検索エンジンの最適化を最大化するためにURLをデザインする方法を確認します。 |
| [ カタログナビゲーション ](navigation.md) | ナビゲーションツールを利用することで、ストア全体を通じて顧客がページ間を移動する方法を定義できます。 |
| [ カタログ検索](search.md) | 顧客がストアフロントで商品を検索するために使用できるツールを確認し、商品カタログに従って検索エクスペリエンスを設定します。 |
| [ カテゴリ ](categories.md) | ストアフロントでの買い物客の体験を最適化できるように、カタログの基本的なカテゴリ構造を確立します。 |
| [製品](products-list.md) | 複数のオプション、スウォッチ、カスタマイズ可能なバンドルを使用して、シンプルな商品から複雑な商品までを作成できます。 ダウンロード可能な商品、サービスやその他の無形商品のバーチャル商品を制作する。 |
| [ カタログの強化](catalog-enrichment.md) | AIが提案した改善点をレビューし、製品名と説明に適用することで、カタログをLLMとAIを活用した検索で正確に表現できます。 |
| [製品属性](product-attributes.md) | 製品オプションの入力コントロールを使用して属性を作成し、製品ページに関する追加情報を提供します。 |

{style="table-layout:auto"}

>[!TIP]
>
>Adobe Commerce B2Bなら、異なる企業向けにカスタマイズされた価格設定を適用して、ゲート化された共有カタログを管理できます。 標準、プライマリ、製品カタログに加えて、異なる価格構造を持つ2種類の共有カタログに顧客がアクセスできます。 共有カタログ機能について詳しくは、[Adobe Commerce B2B ユーザーガイド ](../b2b/catalog-shared.md)を参照してください。
