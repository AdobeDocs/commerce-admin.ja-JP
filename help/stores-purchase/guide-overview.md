# Adobe Commerceストアおよび購入エクスペリエンスガイド

このガイドは、Adobe CommerceおよびMagento Open Source Adminで作業するカスタマーサービス担当者およびセールスマネージャーを対象としています。 ショッピングカートを設定して顧客を支援し、販売を促進する方法など、注文管理と購入機能に関する詳細な情報を提供します。 コア [!DNL Commerce]の設定と機能に関する基本的な理解を前提としています。

主な内容：

| 件名 | 説明 |
| ------- | ----------- |
| [はじめに](introduction.md) | Commerceインスタンス内のセールスおよび購買機能の概要。 |
| [ ストアメニュー](stores-menu.md) | [!UICONTROL Stores] メニューを使用して、ストア階層、構成、在庫、税金、属性を設定するためのツールにアクセスします。 |
| [ ストアとサイト構造](stores.md) | 異なるドメインを持つ複数のweb サイトを持つことができ、各web サイト内に複数のストアを持つことができ、各ストア内に別々のストアビューを持つことができます。 |
| [ ストアビュー](store-views.md) | ストアビューを使用して、ストアを異なるロケールまたは異なるブランディングで利用できるようにします。 |
| [店舗のローカライズ ](store-localize.md) |  |
| [URLを保存](store-urls.md) | 各web サイトについて、ストアフロントに割り当てられるベース URLと、Commerce管理者に割り当てられるURLをカスタマイズします。 |
| [税金](taxes.md) | ストアを設定して、ロケールの要件に従って税金を計算します。これには、検証で計算される付加価値税も含まれます。 |
| [通貨](currency.md) | 商品価格や販売ドキュメントに表示される通貨記号を確認し、店舗やビューごとにカスタマイズできます。 |
| [即時購入](checkout-instant-purchase.md) | ストアでの即時購入を有効にして、アカウントに保存されている情報を使用して、顧客がチェックアウトプロセスを迅速化できるようにします。 |
| [買い物かご](cart.md) | ショッピングカート機能と、顧客購入体験をサポートするための設定方法をご確認ください。 |
| [ チェックアウト ](checkout-process.md) | チェックアウトプロセスと、取引を完了するために必要な情報を収集するためのストアの設定方法を確認します。 |
| [ ギフトカードアカウント ](product-gift-card-workflow.md) | ギフトカードアカウントを使用して、購入したギフトカード商品をストア内の商品の購入に適用する方法を管理します。 |
| [ ウィッシュリスト ](wishlists.md) | 顧客が友人と共有できる商品リストを作成したり、後でカートに転送するために保存したりする機能を提供します。 |
| [支払い方法](payments.md) | チェックアウトや顧客の利便性を向上させるために、ネイティブにサポートされているあらゆる支払い方法やサービスを確認しましょう。 |
| [販売メニュー](sales-menu.md) | [!UICONTROL Sales] メニューを使用して、ワークフロー内の場所に従って注文を操作するためのツールにアクセスします。 |
| [注文管理](orders.md) | 注文が請求書になり、請求書が出荷になるワークフローを通じて、注文の進捗状況とステータスを追跡します。 |
| [発送と配送](delivery.md) | 最適な顧客体験のために、複数の配送方法と配送業者を設定する。 |
