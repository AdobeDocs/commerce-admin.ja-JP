# 通貨

Adobe Commerceは、200以上の国や地域の通貨を受け入れることができます。 ストアが複数の通貨をサポートしている場合、通貨レートが[更新](currency-update.md)されると、_通貨選択_&#x200B;がヘッダーに表示されます。

>[!NOTE]
>
>複数の通貨で支払いを受け付ける場合は、通貨レートの設定を必ず監視してください。変動が利益率に影響を与える可能性があります。

通貨記号は、商品の価格や、注文や請求書などの販売文書に表示されます。 必要に応じて通貨記号をカスタマイズしたり、店舗やビューごとに価格の表示を個別に設定したりできます。
