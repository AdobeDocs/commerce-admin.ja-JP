## 1.3.0(Adobe Commerce 2.3.5 用 )

![新規](../assets/new.svg) **テンプレート** - [!DNL Page Builder] に、既存のコンテンツから作成し、新しいコンテンツ領域に適用できるテンプレートが追加されました。 [!DNL Page Builder] テンプレートには、既存のページ、ブロック、動的ブロック、製品属性、カテゴリの説明のコンテンツとレイアウトの両方が保存されます。 例えば、既存の [!DNL Page Builder] CMS ページをテンプレートとして作成し、そのテンプレートを（そのコンテンツとレイアウトをすべて含めて）適用して、サイトの CMS ページをすばやく作成します。 この新機能については、こちらで説明します。 [テンプレート](templates.md).

![新規](../assets/new.svg) **行、バナー、スライダーのビデオ背景** - [!DNL Page Builder] 行、バナー、スライダーで、背景にビデオを使用できるようになりました。 以下に、これらの新機能について説明します。 [行](row.md), [バナー](banner.md), [スライダー](slider.md).

![新規](../assets/new.svg) **ビデオサポートの追加と機能強化** - [!DNL Page Builder] では、様々なビデオ形式をサポートするようになりました。 YouTubeおよび Vimeo ビデオに加えて、ビデオコンテンツタイプとビデオの背景で、ビデオの URL リンクが次のようなビデオ形式に対応するようになりました。 `.mp4` およびその他のブラウザーでサポートされる形式も含まれます。 また、ビデオコンテンツタイプによって、自動再生機能が追加されます。 以下に、これらの新機能について説明します。 [ビデオコンテンツタイプ](video.md).

![新規](../assets/new.svg) **フルハイトの行、バナー、スライダー** - [!DNL Page Builder] 行、バナー、スライダーは、任意の CSS 単位 (px、%、vh、em) または単位間の計算 (100vh ～ 237px) を使用して、ページの高さを最大に設定できるようになりました。 以下に、これらの新機能について説明します。 [行](row.md), [バナー](banner.md), [スライダー](slider.md).

![新規](../assets/new.svg) **コンテンツタイプアップグレードライブラリ**  — 開発者は、 [!DNL Page Builder] 以前のバージョンとの下位互換性のない問題が発生しないコンテンツタイプ。 このリリース以前は、コンテンツタイプ設定に大幅な変更を加えると、以前に保存したときに表示とデータ損失の問題が発生していました [!DNL Page Builder] コンテンツタイプ。 新しいアップグレードライブラリにより、これらの問題が解消されます。 このライブラリは、新しいバージョンでの設定の変更と一致するように、データベースに保存したコンテンツタイプの以前のバージョンをアップグレードするように設計されています。 [!DNL Page Builder] 新しいリリースの必要に応じて、ネイティブコンテンツタイプでアップグレードライブラリを実行します。 この変更により、 [!DNL Page Builder] 新しいリリースでのコンテンツタイプに対する変更に合わせて、コンテンツタイプは常にアップグレードされます。

>[!IMPORTANT]
>
>保存用に追加のデータベースエンティティを作成した場合 [!DNL Page Builder] コンテンツ、 _必須_ これらのエンティティを `etc/di.xml`. そうしない場合、 [!DNL Page Builder] エンティティに保存されているコンテンツは更新されず、データが失われたり表示に関する問題が発生する可能性があります。 例えば、 [!DNL Page Builder] コンテンツ、ブログエンティティを `etc/di.xml` ファイルを `UpgradableEntitiesPool` アップグレードライブラリで [!DNL Page Builder] ブログで使用されるコンテンツタイプ。 アップグレードライブラリの使用に関する詳細と手順については、 [コンテンツタイプのアップグレード](https://developer.adobe.com/commerce/frontend-core/page-builder/upgrade-content-types/) （内） _Page Builder 開発者ガイド_.
