# 見積もりテンプレートの作成

<!--This topic is linked to from the Commerce Admin quote templates page. If the URL to this topic changes, make sure to add a redirect to prevent the Admin link from returning a 404 error.-->

既存の見積から見積テンプレートを作成して、定期的な注文の大量割引を交渉します。

![管理者から見積テンプレートを作成](./assets/quote-template-create-from-admin.png){width="700" zoomable="yes"}

テンプレートを作成した後、販売者は、注文と数量のしきい値を設定するためのテンプレートオプションを設定し、購入者に送信する前に行項目と見積もりレベルの割引を調整して交渉プロセスを開始できます。

購入者と販売者が合意に達すると、購入者は見積もりテンプレートを受け入れます。 次に、購入者は、見積テンプレートの条件に基づいて、事前承認済みのリンクされた見積もりを[生成できます](account-dashboard-my-quote-templates.md)。 例えば、企業が事業を維持するためにMRO （メンテナンス、修理、運営）注文を受けた場合、購入者または販売者は、見積もりテンプレートを使用して必要な商品を追加し、価格交渉をおこない、月次の定期的な注文の条件を設定できます。 その後、バイヤーは追加の交渉を必要とせずに、リンクされた見積もりを生成し、注文を送信することができます。

詳細については、[見積もりテンプレートの概要](quote-templates-overview.md)を参照してください。
