# アニメーションGIF画像最適化メモ

>[!NOTE]
>
>アニメーションGIF画像の最適化はサポートされていません。 このようなイメージは、イメージの最適化処理の後にアニメーションが失われる場合があります。 この問題を回避するには、アニメーションGIFの画像サイズが、 [[!UICONTROL Media Gallery Image Optimization] 設定](../content-design/media-gallery-image-optimization.md).
