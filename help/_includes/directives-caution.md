# 指令の注意

>[!IMPORTANT]
>
>多数のディレクティブ ( 特に、 _最近表示された項目_ / _比較対象の製品_ 型の場合、ページがキャッシュされていないときに、最初の読み込み時に悪影響が出る可能性があります。 このため、使用する際は注意が必要です。 通常、イシューは作成されません。 しかし、この潜在的な問題について開発チームと話し合う必要があります。
