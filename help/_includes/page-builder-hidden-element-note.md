# [!DNL Page Builder] 非表示の要素

>[!NOTE]
>
>非表示の要素はデータベースに保存され、顧客は見ることができません。 ただし、これらの要素は、サイトをクロールする検索エンジンやその他のボットに対して表示されます。 また、ステージから削除しない限り、非表示の属性を持つ API 呼び出しを通じてリクエストされた場合、コンテンツの一部として返されます。
